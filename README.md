# OtakuDash

**OtakuDash** is a sleek, modern, and feature-rich dynamic dashboard designed for anime enthusiasts. Built with a stunning dark/cyberpunk aesthetic, this application integrates real-time third-party API data, data persistence concepts, full CRUD operations, and beautiful statistical visualizations.

The project is structured flexibly; you can easily switch between custom vanilla CSS or leverage the modern styling power of Tailwind CSS via CDN.

## Features

- **Live Jikan API Fetching:** Automatically pulls real-time top-trending anime data (titles, studios, scores, episodes, and posters) directly from the official MyAnimeList API.
- **Full CRUD Functionality:** Complete management system allowing users to smoothly **Create (Add)**, **Read (Display)**, **Update (Edit)**, and **Delete** anime cards and table records instantly.
- **Interactive Data Visualization:** Integrates dynamic doughnut charts to track your watch status distributions (Active vs. Finished vs. On Hold) smoothly in real-time.
- **Defensive UI & Fallback Systems:** Built-in smart placeholder image handling to protect the UI grid layout from breaking if an entry lacks an image URL.
- **Responsive Fluid Layout:** Fully optimized for mobile, tablet, and desktop viewports using utility-first Tailwind configuration.
- **Sidebar Navigation:** Lightweight hamburger menu navigation built smoothly with vanilla JS without bulky external framework dependencies.
---

## Tech Stack

- **Structure:** HTML5 (Semantic Architecture)
- **Styling:** Tailwind CSS (via CDN) & Custom Vanilla CSS Core
- **Interactivity:** Vanilla JavaScript (ES6+, Async/Await, Array Methods)
- **Data Source:** Jikan RESTful API (v4)
- **Charts:** Chart.js Library
- **Icons:** FontAwesome Icons
